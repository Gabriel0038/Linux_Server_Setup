# Install Ubuntu LTS 24.04

## Objectives

Install a fresh Ubuntu Server 24.04 on a virtual machine - for this I've used Oracle VirtualBox

## Steps

1. Downloaded Ubuntu Server 24.04 LTS from official Ubuntu site
2. Created a new Virtual Machine and used the Ubuntu ISO file to install the OS

3. Assigned resources:
   - RAM: 4 GB
   - CPU: 2 cores
   - Disk: 40 GB
  
The resources were allocated based on the actual and future needs of the project. 

4. Followed installation wizard.
5. Logged in with the initial user created in VB.
6. Completed installation and rebooted system.

## Result

Ubuntu Server installed successfully.

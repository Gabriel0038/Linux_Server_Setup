# Install Ubuntu LTS 24.04

## Objectives

Install a fresh Ubuntu Server 24.04 on a virtual machine - for this I've used Oracle VirtualBox

## Steps

1. Downloaded Ubuntu Server 24.04 ISO.
2. Created a new Virtual Machine.
3. Assigned resources:
   - RAM: (ex: 2 GB)
   - CPU: (ex: 2 cores)
   - Disk: (ex: 25 GB)
4. Booted VM using Ubuntu ISO.
5. Followed installation wizard.
6. Created initial user.
7. Completed installation and rebooted system.

## Result

Ubuntu Server installed successfully.

## Notes

Login tested successfully after reboot.
